# PreviousSubscriptionItemStatus

The status of the previous subscription item after transition.

## Example Usage

```ruby
require "dctest-sdk-ruby"

value = PreviousSubscriptionItemStatus::CANCELED
```


## Values

| Name        | Value       |
| ----------- | ----------- |
| `CANCELED`  | canceled    |
| `ENDED`     | ended       |
| `ABANDONED` | abandoned   |
